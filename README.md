# 🛒 Mercadinho Caparaó

Um sistema de gerenciamento que estou desenvolvendo para o mercado dos meus pais enquanto aprendo programação.

---

## Sobre o projeto

Durante muito tempo eu tentei aprender programação, mas sempre sentia que estava estudando sem direção. Eu fazia alguns exercícios, assistia às aulas, mas não conseguia enxergar um objetivo claro. Muitas vezes eu nem sabia qual seria o próximo assunto para estudar ou como praticar o que tinha aprendido. Então, em vez de estudar para depois pensar em um projeto, decidi escolher um projeto primeiro e deixar que ele guiasse todo o meu aprendizado.

Na mesma época, meus pais comentavam sobre algumas dificuldades que enfrentavam no mercado, principalmente em relação ao controle de estoque e das vendas. Como ainda não utilizam um sistema de gerenciamento, muitas tarefas acabam sendo feitas manualmente.

Esse projeto nasceu então com dois objetivos: aprender desenvolvimento de software de forma prática e, se tudo der certo, criar uma ferramenta que realmente possa facilitar o trabalho deles. Ainda tenho muito para aprender e sei que o sistema vai mudar bastante ao longo do desenvolvimento, mas a ideia é justamente essa. Quero olhar para os primeiros commits daqui a algum tempo e perceber o quanto evoluí.

---

## O que pretendo desenvolver

- 📦 Cadastro de produtos e categorias
- 📊 Controle de estoque
- 🛒 Registro de compras e vendas
- 💰 Controle de caixa
- 👥 Cadastro de fornecedores
- 📈 Relatórios
- 📠 Integração com leitor de código de barras
- 🧾 Impressão de comprovantes

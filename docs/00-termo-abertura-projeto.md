# Termo de Abertura do Projeto

## Cliente

Mercadinho Caparaó

## Desenvolvedora

Mariana Calais

## Objetivo

Desenvolver um sistema de gestão para automatizar o controle operacional e financeiro do Mercadinho Caparaó.

### Premissas

O sistema será utilizado inicialmente em apenas um computador.
O estabelecimento possuirá um leitor de código de barras USB.
O estabelecimento utilizará uma impressora térmica para emissão de comprovantes.
Os produtos serão cadastrados antes de serem vendidos.

### Restrições

A primeira versão não emitirá nota fiscal eletrônica.
O sistema será desenvolvido para apenas um estabelecimento.
O acesso via celular não fará parte da primeira versão.
O sistema deverá funcionar em computadores com Windows 11.

### Critérios de Sucesso

Registrar vendas utilizando leitor de código de barras.
Atualizar automaticamente o estoque.
Controlar entradas e saídas de mercadorias.
Emitir comprovantes de venda.
Permitir consultas de estoque e lucro.

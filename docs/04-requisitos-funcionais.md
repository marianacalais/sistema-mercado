# Requisitos Funcionais

## RF01 - Produtos

**RF01.01** O sistema deverá permitir cadastrar um produto.

**RF01.02** O sistema deverá permitir alterar os dados de um produto.

**RF01.03** O sistema deverá permitir consultar os dados de um produto.

**RF01.04** O sistema deverá permitir listar todos os produtos.

**RF01.05** O sistema deverá permitir desativar um produto.

**RF01.06** O sistema deverá permitir localizar um produto pelo código de barras.

**RF01.07** O sistema deverá permitir localizar um produto pelo nome.

**RF01.08** O sistema deverá permitir associar um produto a uma categoria.

---

## RF02 - Categorias

**RF02.01** O sistema deverá permitir cadastrar categorias.

**RF02.02** O sistema deverá permitir alterar categorias.

**RF02.03** O sistema deverá permitir consultar os dados de uma categoria.

**RF02.04** O sistema deverá permitir listar categorias.

**RF02.05** O sistema deverá permitir desativar categorias.

---

## RF03 - Fornecedores

**RF03.01** O sistema deverá permitir cadastrar fornecedores.

**RF03.02** O sistema deverá permitir alterar fornecedores.

**RF03.03** O sistema deverá permitir consultar os dados de um fornecedor.

**RF03.04** O sistema deverá permitir listar fornecedores.

**RF03.05** O sistema deverá permitir desativar fornecedores.

---

## RF04 - Compras

**RF04.01** O sistema deverá permitir registrar uma compra.

**RF04.02** O sistema deverá permitir adicionar produtos à compra, informando a quantidade comprada.

**RF04.03** O sistema deverá permitir remover produtos da compra, informando a quantidade a ser removida.

**RF04.04** O sistema deverá permitir associar a compra a um fornecedor.

**RF04.05** O sistema deverá permitir consultar compras realizadas.

---

## RF05 - Estoque

**RF05.01** O sistema deverá registrar entradas de produtos no estoque.

**RF05.02** O sistema deverá registrar saídas de produtos do estoque.

**RF05.03** O sistema deverá permitir realizar ajustes de estoque.

**RF05.04** O sistema deverá permitir registrar perdas de produtos.

**RF05.05** O sistema deverá permitir consultar a quantidade disponível de um produto.

**RF05.06** O sistema deverá permitir consultar produtos abaixo do estoque mínimo ou sem estoque.

**RF05.07** O sistema deverá permitir consultar o histórico de movimentações do estoque.

---

## RF06 - Clientes

**RF06.01** O sistema deverá permitir cadastrar clientes.

**RF06.02** O sistema deverá permitir alterar clientes.

**RF06.03** O sistema deverá permitir consultar os dados de um cliente.

**RF06.04** O sistema deverá permitir listar clientes.

**RF06.05** O sistema deverá permitir consultar o histórico de compras de um cliente.

**RF06.06** O sistema deverá permitir consultar os débitos pendentes de um cliente.

**RF06.07** O sistema deverá permitir registrar o pagamento de um débito.

---

## RF07 - Vendas

**RF07.01** O sistema deverá permitir iniciar uma venda.

**RF07.02** O sistema deverá permitir adicionar produtos a uma venda em andamento, informando a quantidade a ser adicionada.

**RF07.03** O sistema deverá permitir remover produtos de uma venda em andamento, informando a quantidade a ser removida.

**RF07.04** O sistema deverá calcular automaticamente o valor da venda.

**RF07.05** O sistema deverá permitir aplicar descontos.

**RF07.06** O sistema deverá registrar a forma de pagamento.

**RF07.07** O sistema deverá permitir registrar pagamentos utilizando múltiplas formas de pagamento.

**RF07.08** O sistema deverá concluir a venda.

**RF07.09** O sistema deverá cancelar uma venda.

**RF07.10** O sistema deverá consultar uma venda realizada.

**RF07.11** O sistema deverá emitir o comprovante da venda.

---

## RF08 - Caixa

**RF08.01** - O sistema deverá registrar a abertura do caixa.

**RF08.02** - O sistema deverá registrar o fechamento do caixa.

**RF08.03** - O sistema deverá registrar movimentações financeiras do caixa.

**RF08.04** - O sistema deverá registrar retiradas de dinheiro do caixa.

**RF08.05** - O sistema deverá registrar despesas pagas pelo caixa.

**RF08.06** - O sistema deverá apresentar o resumo financeiro do caixa ao final do expediente.

**RF08.07** - O sistema deverá permitir consultar o histórico dos fechamentos de caixa.

**RF08.08** O sistema deverá permitir consultar o saldo atual do caixa.

**RF08.09** O sistema deverá permitir registrar sangrias do caixa.

---

## RF09 - Relatórios

**RF09.01** - O sistema deverá gerar relatório de estoque.

**RF09.02** - O sistema deverá gerar relatório de vendas por período.

**RF09.03** - O sistema deverá gerar relatório de compras por período.

**RF09.04** - O sistema deverá gerar relatório de produtos com estoque baixo.

**RF09.05** - O sistema deverá gerar relatório de produtos mais vendidos.

**RF09.06** - O sistema deverá gerar relatório de lucro obtido com as vendas por período.

**RF09.07** - O sistema deverá gerar relatório de clientes com pagamentos pendentes.

**RF09.08** - O sistema deverá gerar relatório do histórico financeiro do caixa por período.

**RF09.09** O sistema deverá gerar relatório de produtos sem estoque.

**RF09.10** O sistema deverá gerar relatório de movimentações de estoque.
